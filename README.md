# 💻 Programação Front-end

Bem-vindo(a) ao meu repositório de **Programação Front-end**! 🚀

Este espaço foi criado para armazenar, organizar e documentar todo o meu progresso, exercícios práticos e projetos desenvolvidos durante a disciplina. O objetivo aqui é aplicar os conceitos teóricos na prática e construir uma base sólida no desenvolvimento de interfaces web.

---

## 🎯 Objetivo da Disciplina

Compreender e aplicar as principais tecnologias e boas práticas para a construção de páginas web responsivas, acessíveis e interativas, separando corretamente a estrutura, a estilização e o comportamento.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

Durante os estudos, as seguintes tecnologias estão sendo exploradas:

*   **HTML5:** Estruturação semântica das páginas.
*   **CSS3:** Estilização, layouts (Flexbox/Grid) e design responsivo.
*   **JavaScript (ES6+):** Lógica de programação, manipulação do DOM e uso de APIs (como o Canvas).
*   **VS Code:** Editor de código principal.

---

## 📂 Estrutura do Repositório

O repositório está organizado de forma a facilitar a busca por tópicos específicos estudados em aula:

*   📁 `aulas/`: Pequenos exercícios e códigos feitos durante as explicações.
*   📁 `projetos/`: Trabalhos maiores e mais completos.
*   📁 `desafios/`: Resoluções de desafios propostos pelo professor.
*   📄 `anotacoes.md`: Resumos de conceitos importantes (opcional).

*(Sinta-se à vontade para adaptar essa estrutura conforme a sua necessidade ao longo do curso)*
